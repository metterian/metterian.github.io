🧐 대체 시그모이드(Sigmoid) 함수가 뭔데?

Jekyll also offers powerful support for code snippets: like these

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
